Vitality Index (Mobile)

Largest Contentful Paint (LCP)

What, Why, How?

What: Time for largest content to load. Why: Slow LCP feels laggy. How: Optimize images/JS. More: Preload heroes, use CDN for 20-50% gains.

Interaction to Next Paint (INP)

What, Why, How?

What: Input responsiveness. Why: High INP frustrates users. How: Reduce JS tasks. More: Break tasks <50ms, web workers for +30% speed.

Cumulative Layout Shift (CLS)

What, Why, How?

What: Unexpected shifts. Why: Causes misclicks. How: Set sizes. More: Aspect ratios on media prevent 80% shifts.
